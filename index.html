Schéma Système de Domotique - Cyberdéfense

Un système intégré pour la sécurité et l'automatisation domestique

Firewall Principal

Fonctions : IDS/IPS (Suricata ou Zeek), Inspection TLS (MITM pour IoT uniquement), Segmentation stricte des VLANs (IoT, Admin, Caméras, Services, Guest), Geo-blocking et filtrage DNS (Unbound, DNS over TLS), Fail2Ban + Cloudflare Zero Trust Access possible.

VLAN Management

Services : Home Assistant (automatisations locales, contrôle radar, portes, éclairage), Reconnaissance faciale (DeepStack, CompreFace), Serveur Biométrique (Iris + Empreintes), Agent de surveillance Mosquitto (MQTT Broker sécurisé), Node-RED (scénarios complexes), Draggable API (interactions utilisateur).

VLAN Caméras

Équipements : Caméras EO/IR (thermiques + optiques), Caméras CCTV (PoE, RTSP, ONVIF), Radar de mouvement longue portée (sous forêt/pluie), Microphones d'ambiance pour levée de doute (stockage local uniquement).

VLAN IoT

Équipements : Portes biométriques avec lecteur passif, Interphone vidéo Ubiquiti intégré Home Assistant, Détecteurs de fumée, gaz, humidité, Prises connectées, thermostats, volets, etc. Sécurité : Chaque device n'a accès qu’au MQTT Broker, Aucune communication directe entre devices, Surveillance de comportement.

VLAN Services

Services locaux : Jellyfin / Jellyseer (médias privés), Plex (accès distant avec tunneling chiffré), Nextcloud + OnlyOffice (auto-hébergé), Affine / Joplin Server / Logseq (notes structurées, to-do sécurisée), Vaultwarden (gestionnaire de mots de passe local), Uptime Kuma (monitoring des services), Grafana + Prometheus (visualisation système et capteurs), Paperless NGX (archivage intelligent de documents), Minio / S3-compatible NAS (stockage objet privé), Syncthing / Resilio (synchro P2P auto avec backup différé).

VLAN Admin

Matériel : Proxmox (7995WX + 4080 Super), VM Docker (Traefik, Authelia pour authentification centralisée avec MFA), Watchtower (maj auto), Portainer / Yacht (gestion UI des conteneurs), OpenLDAP / Keycloak (authentification centralisée).
